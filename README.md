
========

This is my Neovim editor setup.
Feel free to fork it
and submit a pull request if you found any bug.

Instalation
-----------

    $ git clone https://github.com/kris89/neovim-config.git ~/neovim-config
    $ cd ~/neovim-config
    $ chmod +x ./install.sh
    $ ./install.sh
    $ nvim
    $ :PlugInstall
    $ :UpdateRemotePlugins
